# 湖北恩施五日行程 App

根据《湖北5日游攻略-合并版-住宿更新.docx》整理的 iPhone 可安装 PWA，覆盖 2026 年 8 月 5 日至 9 日行程。

## 主要内容

- 按日期查看五天时间线、注意事项、酒店名称和建议价格区间
- 14 项交通、住宿、门票预订及 48 小时复核清单
- 宣恩、屏山、佛宝山、腾龙洞及返程异常的备选方案
- 佛宝山“资格、开漂、批次、车辆”四条件执行闸门
- 16 项证件、山路、漂流、洞穴和药品行李清单
- 勾选状态和每日补充记录保存在当前设备浏览器中
- 行程正文和 8 张景点图片可离线读取

动态时刻、价格、场次和开放状态不是订单凭证，应以最终订单、12306、景区电话及当天公告为准。

## 本地运行

```powershell
npm install
npm run dev
```

生产构建和局域网预览：

```powershell
npm run build
npm run preview -- --host 0.0.0.0 --port 4173
```

构建产物位于 `dist/`，可部署到任意支持 HTTPS 的静态网站服务。

## 安装到 iPhone

1. 将 `dist/` 部署到稳定的 HTTPS 地址。
2. 用 iPhone 的 Safari 打开该地址并等待首屏完整加载。
3. 点击 Safari“分享”，选择“添加到主屏幕”。
4. 从主屏幕打开一次 App，随后可在断网状态查看已缓存的行程和图片。

局域网 `http://电脑IP:4173/` 适合预览，但普通 HTTP 不能保证 iPhone Service Worker 和离线安装正常工作。正式使用应采用 HTTPS。

## 数据与隐私

- 行程日期、酒店和勾选状态不会主动上传；勾选与备注只存于 Safari 本机。
- 清除 Safari 网站数据、删除对应站点存储或更换网址后，本机勾选与备注会丢失。
- 本应用包含具体旅行日期和住宿安排，公开部署后任何获得网址的人都能查看。
- 景点图片来自原 Word 文档，仅用于本次行程规划；公开发布或商业使用需另行确认授权。
